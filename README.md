## Présentation du Projet
Ce projet est une application simple en Java pour gérer les présences des étudiants. Elle permet d'ajouter des étudiants et d'enregistrer leur présence. L'objectif est d'appliquer les concepts de Java et de développement backend appris en cours.

## Objectifs du Projet
- Créer une application en Java pour la gestion des étudiants.
- Permettre l'enregistrement des présences.
- Appliquer les notions de base de Java et du backend.

## Technologies Utilisées
- Langage : Java
- Environnement : JDK (Java Development Kit)

## Structure du Projet
Le projet est organisé de manière simple :
- `backend_java/` : Contient le code source Java.
-'frontend_flutter/' : contient le code source dart
- `README.md` : Ce fichier de documentation.

## Comment Exécuter le Projet
Pour lancer l'application, suivez ces étapes simples :
1. Assurez-vous que Java est installé sur votre ordinateur.
2. Ouvrez un terminal dans le dossier du projet.
3. Compilez le code avec la commande : `javac backend_java*.java`
4. Exécutez l'application avec : `java -cp backend_java Main`

## Conclusion
Ce projet permet de mettre en pratique les bases du développement en Java. Il offre une introduction aux applications backend simples.
